# ¿Qué es CSS?

CSS, o Hojas de Estilo en Cascada, es un lenguaje de hojas de estilo utilizado para controlar la presentación y el diseño de documentos HTML. Permite aplicar estilos a las páginas web, incluyendo aspectos de diseño como colores, fuentes y espaciado.

## Sintaxis Básica

Las reglas CSS están formadas por un selector y un conjunto de propiedades.

**Ejemplo:**

```css showLinesNumbers
selector {
  propiedad: valor;
}
```

```css
h1 {
  color: blue;
  font-size: 24px;
}
```

## Selectores

Los selectores se utilizan para dirigirse a elementos HTML para su estilización. Los selectores comunes incluyen:

### Selector de Elemento

Dirige a los elementos por su nombre de etiqueta.

```css
p {
  color: green;
}
```

### Selector de Clase

Dirige a los elementos con un atributo de clase específico.

```css
.mi-clase {
  font-style: italic;
}
```

### Selector de ID

Dirige a un elemento único con un ID específico.

```css
#mi-id {
  margin: 10px;
}
```

## Técnicas de Diseño

CSS proporciona varios métodos para crear diseños:

### Flexbox

Un modelo de diseño para distribuir espacio en un contenedor

```css
.contenedor {
  display: flex;
  justify-content: center;
}
```

### Grid

Un sistema de diseño para crear diseños basados en cuadrículas complejas.

```css
.contenedor-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
```

## Conclusión

CSS es esencial para estilizar y maquetar páginas web. Al dominar los fundamentos de los selectores, propiedades y técnicas de diseño, puedes crear sitios web visualmente atractivos y bien estructurados.
